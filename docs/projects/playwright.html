How to run

Each project has its own package.json. Install and run from the project folder:

cd Playwright/PROJECTS/PROJECT_01_Login_Logout_Testing
npm install
npx playwright test

With UI or headed browser:

npx playwright test --ui
npx playwright test --headed

See each project’s README for prerequisites (e.g. base URL, test accounts) and env setup. No credentials in repo.
